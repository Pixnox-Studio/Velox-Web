Velox

A simple yet flexible ModernGL + Pygame framework for game development and rapid prototyping.

🎮 Introduction

Velox is a Python game engine designed with simplicity, clarity, and modularity in mind. It uses ModernGL for rendering and Pygame for window management and input. Ideal for small games, tools, and graphics experiments.

📦 Installation

pip install VeloxEngine

Make sure you have Python 3.7+ installed.

🖥️ Creating a Window

import VeloxEngine

engine = Engine()
engine.run()

This initializes a resizable OpenGL window and enters the game loop.

🔁 Engine Structure

class Engine:
    def run(self):
        while self.running:
            self.event_handler()
            self.update()
            self.render()
        self.close()

The Engine class handles initialization, the main loop, and graceful shutdown.

🎬 Scene Management

from velox.scenes.scene import Scene

class MyScene(Scene):
    def on_create(self):
        print("Scene started")

    def on_update(self):
        pass

engine.add_scene("main", MyScene())

Scenes are registered with the engine and are responsible for update, rendering, and events.

🎯 Event Handling

def on_event(self, event):
    if event.type == pg.KEYDOWN:
        if event.key == pg.K_ESCAPE:
            self.engine.running = False

Events are passed directly to the active scene for handling.
