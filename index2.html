Velox

A blazing-fast SDL2 wrapper for Python. Built for speed, flexibility, and control.

ğŸ® Introduction

Velox is a low-level yet beginner-friendly Python library built on top of SDL2. It provides game developers with efficient access to rendering, input, and sound, with minimal abstraction.
