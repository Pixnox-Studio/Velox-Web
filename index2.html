Velox

A blazing-fast SDL2 wrapper for Python. Built for speed, flexibility, and control.

🎮 Introduction

Velox is a low-level yet beginner-friendly Python library built on top of SDL2. It provides game developers with efficient access to rendering, input, and sound, with minimal abstraction.

📦 Installation

pip install velox

Requires Python 3.7+ and SDL2 installed on your system.

🖥️ Creating a Window

import velox

win = velox.Window("Velox Demo", 800, 600)
win.show()

🔁 Game Loop

running = True
while running:
    for event in velox.poll_events():
        if event.type == velox.QUIT:
            running = False

    win.clear(0, 0, 0)  # Clear with black
    # drawing code...
    win.present()       # Show frame

🖌️ Drawing

win.draw_rect(100, 100, 50, 50, color=(255, 0, 0))  # Red square

Velox supports basic shapes, images, and text rendering via SDL2 surfaces and textures.

🎯 Event Handling

for event in velox.poll_events():
    if event.type == velox.KEYDOWN:
        print("Key pressed:", event.key)
