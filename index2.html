Velox

A blazing-fast SDL2 wrapper for Python. Built for speed, flexibility, and control.

🎮 Introduction

Velox is a low-level SDL2 wrapper for Python, giving you full control over rendering, events, and frame timing. It's designed for developers who want performance and precision without the overhead of large frameworks.

📦 Installation

pip install pysdl2

Requires Python 3.7+ and SDL2 installed on your system.

🖥️ Creating a Window

import velox

surface = velox.display_set_mode((800, 600))
velox.display_set_caption("Velox Demo")

🔁 Game Loop

running = True
clock = velox.Clock()

while running:
    for event in velox.event_get():
        if event.type == velox.QUIT:
            running = False

    surface.paint((0, 0, 0))  # Clear with black
    # drawing code here...
    velox.display_flip()
    clock.tick(60)

🖌️ Drawing

# Create a red surface
red_surface = velox.Surface(50, 50)
red_surface.paint((255, 0, 0))

# Draw it onto the screen at position (100, 100)
surface.flash(red_surface, (100, 100))

Use Surface.paint() to color surfaces and flash() to render them onto the display.

🎯 Event Handling

keys = velox.key_get_pressed()
if keys.get(velox.K_w):
    print("W is being held down")

Use event_get() for SDL events and key_get_pressed() for real-time key state polling.

⏱️ Frame Limiting

clock = velox.Clock()

while True:
    fps = clock.tick(60)
    print("FPS:", fps)

Use Clock.tick() to limit frame rate and track current FPS.
